# 🛒 Sistema de Ventas - Laravel

Aplicación web desarrollada en Laravel para la gestión de ventas, productos y clientes.

## 🚀 Tecnologías
- Laravel (PHP)
- MySQL
- Blade / Vite
- Bootstrap

## 📦 Funcionalidades
- Gestión de productos
- Registro de ventas
- Administración de clientes
- Control de inventario
- Reportes básicos

## 📁 Estructura
- app/ → lógica del sistema
- routes/ → rutas web
- resources/ → vistas
- database/ → migraciones y seeds

## ⚙️ Instalación

```bash
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
🔐 Configuración

Configurar base de datos en el archivo .env
